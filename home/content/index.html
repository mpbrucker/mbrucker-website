Experience

Dwolla

InfoSec Intern | Summer 2016

In a summer internship for Des-Moines, Iowa based financial tech startup Dwolla, I helped strengthen their cybersecurity platform and procedures. I tested and helped deploy a number of different open-source security tools - file integrity monitoring software, XSS testing, host scanning, etc. - to help improve Dwolla's automated threat monitoring and alerting processes, while also developing a number of Python-based programs to help improve the security infrastructure, which included improving security groups with AWS (Amazon Web Services) and performing automated security scanning.
